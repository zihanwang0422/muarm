## 环境准备

```bash
cd /home/wzh/manipulation_mujoco
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
```

## kinematic 运行命令

```bash
cd /home/wzh/manipulation_mujoco
source .venv/bin/activate

python kinematic/run_fk.py
python kinematic/run_ik.py
python kinematic/run_trajectory.py
```

## RL 训练命令

```bash
cd /home/wzh/manipulation_mujoco
source .venv/bin/activate

python learning/panda_rl/train.py --mode train --family rl --task reach --algo ppo --timesteps 300000
python learning/panda_rl/train.py --mode train --family rl --task reach --algo sac --timesteps 300000
python learning/panda_rl/train.py --mode train --family rl --task push --algo td3 --timesteps 300000
python learning/panda_rl/train.py --mode train --family rl --task pick_place --algo ddpg --timesteps 300000
```

## RL 评估命令

```bash
cd /home/wzh/manipulation_mujoco
source .venv/bin/activate

python learning/panda_rl/train.py --mode eval --task reach --algo ppo --model-path learning/panda_rl/runs/reach_ppo.zip --episodes 5
python learning/panda_rl/train.py --mode eval --task push --algo sac --model-path learning/panda_rl/runs/push_sac.zip --episodes 5
```

## IL(BC) 训练与评估命令
