# manipulation_mujoco

## kinematic жЁЎеқ—жҖқз»ҙеҜјеӣҫ

```text
kinematic
в”ңв”Җв”Җ PandaKinematics (class жҺҘеҸЈ)
в”Ӯ   в”ңв”Җв”Җ build_from_mjcf
в”Ӯ   в”ңв”Җв”Җ fk
в”Ӯ   в”ңв”Җв”Җ ik
в”Ӯ   в””в”Җв”Җ jacobian
в”ңв”Җв”Җ TrajectoryGenerator (class жҺҘеҸЈ)
в”Ӯ   в”ңв”Җв”Җ cubic / multi_waypoint
в”Ӯ   в”ңв”Җв”Җ catmull_rom  вҶҗ иҝһз»ӯй—ӯеҗҲиҪЁиҝ№ (8еӯ—еҪў)
в”Ӯ   в”ңв”Җв”Җ cartesian_linear
в”Ӯ   в””в”Җв”Җ cartesian_arc
в”ңв”Җв”Җ run_fk.py
в”ңв”Җв”Җ run_ik.py
в””в”Җв”Җ run_trajectory.py  вҶҗ 8еӯ—еҪў Lissajous иҪЁиҝ№жј”зӨә
```

## learning жЁЎеқ—жҖқз»ҙеҜјеӣҫ

```text
learning/
в”ңв”Җв”Җ envs/
в”Ӯ   в”ңв”Җв”Җ base_env.py          вҶҗ MuJocoRobotEnv йҖҡз”Ёеҹәзұ» (robot-agnostic)
в”Ӯ   в”ңв”Җв”Җ registry.py          вҶҗ make_env("panda"/"so_arm", "reach"/"push"/"pick_place")
в”Ӯ   в””в”Җв”Җ tasks/
в”Ӯ       в”ңв”Җв”Җ base_task.py     вҶҗ BaseTask жҺҘеҸЈеҚҸи®®
в”Ӯ       в”ңв”Җв”Җ reach.py
в”Ӯ       в”ңв”Җв”Җ push.py
в”Ӯ       в””в”Җв”Җ pick_place.py
в”ңв”Җв”Җ robots/
в”Ӯ   в”ңв”Җв”Җ panda.py             вҶҗ FrankaPandaEnv (4D з¬ӣеҚЎе°”еўһйҮҸ + еӨ№зҲӘ)
в”Ӯ   в””в”Җв”Җ so_arm.py            вҶҗ SoArm100Env   (6D е…іиҠӮйҖҹеәҰ)
в”ңв”Җв”Җ algos/
в”Ӯ   в”ңв”Җв”Җ rl_trainer.py        вҶҗ train_rl (PPO/SAC/TD3/DDPG + SuccessRateCallback)
в”Ӯ   в””в”Җв”Җ il/
в”Ӯ       в””в”Җв”Җ bc.py            вҶҗ Behavior Cloning (MLP + cosine LR)
в”ңв”Җв”Җ utils/
в”Ӯ   в””в”Җв”Җ visualize.py         вҶҗ MuJoCo overlay helpers
в”ңв”Җв”Җ train.py                 вҶҗ и®ӯз»ғе…ҘеҸЈ
в””в”Җв”Җ play.py                  вҶҗ д»ҘжӯЈеёёйҖҹеәҰеҸҜи§ҶеҢ–еӣһж”ҫи®ӯз»ғеҘҪзҡ„зӯ–з•Ҙ
```

## RL и®ӯз»ғе‘Ҫд»Ө

```bash
cd /home/wzh/manipulation_mujoco
source .venv/bin/activate

# --- Headless (еҝ«йҖҹ, жҺЁиҚҗи®ӯз»ғ) ---
python learning/train.py --robot panda --task reach    --algo sac  --timesteps 200000
python learning/train.py --robot panda --task push     --algo td3  --timesteps 300000
python learning/train.py --robot panda --task pick_place --algo sac --timesteps 500000

# --- ејҖеҗҜе®һж—¶еҸҜи§ҶеҢ– (дјҡйҷҚдҪҺи®ӯз»ғйҖҹеәҰ) ---
python learning/train.py --robot panda --task reach --algo sac --timesteps 200000 --render

# --- еӨҡе№¶иЎҢ env (д»… headless) ---
python learning/train.py --robot panda --task reach --algo sac --timesteps 200000 --n-envs 4

# --- TensorBoard ж—Ҙеҝ— ---
python learning/train.py --robot panda --task reach --algo sac --timesteps 200000 --tensorboard
tensorboard --logdir learning/runs/tb

# --- SO-ARM100 ---
python learning/train.py --robot so_arm --task reach --algo sac --timesteps 100000
```

## Play вҖ“ д»ҘжӯЈеёёйҖҹеәҰеҸҜи§ҶеҢ–зӯ–з•Ҙ

```bash
cd /home/wzh/manipulation_mujoco
source .venv/bin/activate

# ж’ӯж”ҫ RL зӯ–з•Ҙ (SAC reach)
python learning/play.py --robot panda --task reach --algo sac \
    --model learning/runs/panda_reach_sac.zip --episodes 5

# ж’ӯж”ҫ BC зӯ–з•Ҙ
python learning/play.py --robot panda --task reach --algo bc \
    --model learning/runs/panda_reach_bc.pt --episodes 5

# Push / Pick-Place
python learning/play.py --robot panda --task push      --algo sac \
    --model learning/runs/panda_push_sac.zip
python learning/play.py --robot panda --task pick_place --algo sac \
    --model learning/runs/panda_pick_place_sac.zip

# SO-ARM100
python learning/play.py --robot so_arm --task reach --algo sac \
    --model learning/runs/so_arm_reach_sac.zip
```

## жү©еұ•ж–°жңәжў°иҮӮ

1. еңЁ `learning/robots/` ж–°е»ә `my_robot.py`пјҢз»§жүҝ `MuJocoRobotEnv`
2. е®һзҺ° `_build_spaces / _get_obs / _apply_action / get_ee_pos / get_ee_body`
3. еңЁ `learning/envs/registry.py` зҡ„ `make_env` дёӯж·»еҠ  `elif robot_key == "my_robot":`
4. еҚіеҸҜдҪҝз”ЁзӣёеҗҢзҡ„ `train.py / play.py` е‘Ҫд»Ө
